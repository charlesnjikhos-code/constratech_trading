## 🚀 HOW TO USE YOUR CMS

### Step 1: Install (One-Time Setup)
1. Start XAMPP (Apache + MySQL)
2. Go to: `http://localhost/constratech_trading/setup.php`
3. Enter database details:
   - Host: `localhost`
   - Username: `root`
   - Password: *(leave blank)*
   - Database: `constratech_cms`
4. Click **"Install Database"**
5. **Delete `setup.php`** after successful installation

### Step 2: Login to CMS
1. Go to: `http://localhost/constratech_trading/cms-login.html`
2. Login with:
   - **Username:** `admin`
   - **Password:** `constratech2025`

### Step 3: Edit Your Website Content
1. Click any section in the sidebar:
   - Company Information
   - Hero Section (Homepage banner)
   - About Section
   - Services
   - Portfolio (Gallery)
   - Contact Information
   - Footer Settings
2. Make your changes
3. Click **"Save Changes"** (or "Save All Changes")

### Step 4: View Your Changes
- Go to your website: `http://localhost/constratech_trading/index.html`
- All changes appear immediately!
